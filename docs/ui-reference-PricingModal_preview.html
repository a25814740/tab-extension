個人版設計原則

維持低價格的關鍵，是把個人版定位成單人整理工具：資料主存於本地與使用者自己的 Google Drive，避免過早承擔多人協作、權限與即時同步成本。

團隊 / 企業版設計原則

一旦進入共享 Workspace、多人編輯、角色權限與活動紀錄，主資料應轉移到自家後端。Google Drive 可作為升級前匯入來源，或作為企業受控備份，而非一般成員自由備份空間。
