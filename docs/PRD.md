# PRD

This extension targets heavy tab users who need structured spaces, folders, collections, and collaboration.

## Goals

- Reduce tab overload
- Provide quick save/restore workflows
- Support sharing and collaboration
- Offer AI-driven organization

## Pricing / Plans (現況)

- Trial：60 天全功能，期滿需升級
- Personal（年費）：目前 UI 亮顯的付費主力
- Enterprise：客製報價，占位用
- 尚未實作的項目：PAYUNi API 串接、Pro 月付方案卡片 / 比較表（僅在程式碼中預留 plan id）
